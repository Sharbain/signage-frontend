# Signage Frontend (Vercel)

## Run locally
1. `npm install`
2. Create `.env` from `.env.example`
3. `npm run dev`

## Deploy on Vercel
- Framework: Vite
- Build command: `npm run build`
- Output directory: `client/dist`
- Environment variables:
  - `VITE_API_BASE_URL` = `https://YOUR-RENDER-BACKEND.onrender.com/api`
